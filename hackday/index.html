Nashville hack day

a free one-day community event by Nashville developers for Nashville developers

On August 11, 2012, eleven speakers gave short tech talks to a packed house at centresource. We had lunch catered in. We chatted, hacked on IRC bots, debugged bash scripts, did Ruby koans, and tried to explain git to each other. It was a blast.

Thank you to everyone who came out.

All the talks - slides, links, source code, and more.
